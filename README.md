# index-ufc-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态资源形式存放，不针对任何特定域名或网站，仅作为通用页面集合。

## 项目简介

index-ufc-lab 是一个轻量级的 HTML 页面归档仓库。项目内包含若干独立编写或整理的 HTML 文件，每个文件均可在浏览器中直接打开并运行。仓库本身不依赖后端服务或数据库，所有内容均为纯前端静态页面。

## 目录说明

```
/
├── README.md          # 本文件
├── index.html         # 仓库入口页面（可选）
├── pages/             # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 公共资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：包含所有独立 HTML 页面，每个页面相对独立，可单独访问。
- `assets/`：存放页面共用的样式、脚本或图片资源，便于统一管理。

## 页面归档说明

- 每个 HTML 页面均为独立归档，不依赖其他页面。
- 页面内容可能包含实验性功能、学习示例或工具页面，不保证长期可用或更新。
- 部分页面可能引用外部资源（如 CDN），访问时需保持网络通畅。

## 维护说明

- 本仓库不定期更新，新增或修改页面会直接提交至 `main` 分支。
- 欢迎提交 Issue 或 Pull Request，但请确保内容符合仓库定位（独立 HTML 页面归档）。
- 不接受包含恶意代码、广告或侵犯第三方权益的内容。
- 仓库维护者保留对页面的增删改权利。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中直接打开 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后访问。
